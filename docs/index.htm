jQuery changeElementType Plugin Examples

  • Basic element change
  • Multiple element change
  • Event persistence
  • Nested elements

Basic element change (div to p)

Hello, my name is Simon.
      
        $('#example-basic-div').changeElementType('p');

Multiple element change (divs to ps)

Hello, my name is Simon.
And I like to do drawings.
      
        $('.example-multiple-div').changeElementType('p');

Event persistence

I have a click event attached to me.
      
        $('#example-events-div').click(function(e) {
          console.log("I am a " + e.target.tagName);
        })
        .changeElementType('p');

Nested

      
        var select = $('#example-nested-select');
        select.children().changeElementType('div');
        select.changeElementType('div');
